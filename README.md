# GPU Product Showcase

This project displays a list of GPU products with a sorting feature.

## Features
- **Product Cards**: Displays image, title, description, price, and stock status.
- **Sorting**: Sort products by price (Low to High / High to Low).
- **Responsive Design**: Adapts to different screen sizes.

## Files
- `index.html`: Main structure.
- `style.css`: Styling and layout.
- `script.js`: Logic for sorting functionality.
